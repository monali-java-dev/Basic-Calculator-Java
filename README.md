# Basic Calculator (Java)

## 📌 Description
This is a simple Java console-based calculator that performs basic arithmetic operations.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division (with zero handling)

## 🛠️ Technologies Used
- Java
- Scanner Class
- Conditional Statements

## ▶️ How to Run
1. Open project in Eclipse
2. Run the Java file
3. Enter numbers and choose operation
